1. Authentication/Login/permissions
2. Categories/Events (Similar to Catalog)
3. Create Event
4. Cart Service (Add to the cart)
5. Payment service
6. Search (by price)
7. Help
8. Delete Event
9. Create User account

10. Admin API authentication (user information)
